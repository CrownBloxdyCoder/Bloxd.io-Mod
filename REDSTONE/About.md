# About
Creator:lincolnearley

Some redstone stuff I've been working on.
PISTONS:
Upward pistons are engraved stone
Downward pistons are engraved andesite
All horizontal pistons are empty mailboxes with the lip facing the direction you want to extend
STICKY PISTONS:
Upward sticky pistons are Emerald Blocks (codeblock needed to obtain)
Downward sticky pistons are Green Bricks (codeblock needed to obtain)
All horizontal sticky pistons are mailboxes with a book in them
REDSTONE DUST = RED CARPET
PRESSURE PLATE = BROWN CARPET
REDSTONE BLOCK = FIREBALL BLOCK (lil' glitchy)
BULB = DIM LAMP OFF
